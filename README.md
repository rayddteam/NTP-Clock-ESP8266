
# ABRobot Kyiv Clock (ESP32-C3 + 0.42" OLED)

- Щохвилини: wake → оновлення екрана → deep sleep
- Раз на 30 хв: Wi‑Fi + NTP синхронізація
- Вивід: `ГГ:ХХ` великим шрифтом. Якщо **минулА** NTP‑синхронізація **невдала**, між годинами і хвилинами ставиться `.` замість `:`. Нижній рядок: `Таймзона: Київ`.

## Збірка (PlatformIO)

1. Відкрийте папку в PlatformIO/VS Code.
2. Заповніть `WIFI_SSID` та `WIFI_PASSWORD` у `src/main.cpp`.
3. Прошийте плату `ESP32-C3` (профіль `esp32-c3-devkitm-1`).

I2C шина для вбудованого OLED: `SDA=GPIO5`, `SCL=GPIO6`.
